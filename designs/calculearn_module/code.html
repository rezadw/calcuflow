The Power Rule

The Power Rule is a quick and foundational method for finding the derivative of functions where the variable is raised to a constant power. It dramatically simplifies differentiation.

lightbulb Concept

Instead of using the lengthy limit definition of a derivative, the power rule provides a shortcut. When you have a function in the form of f(x) = x^n, you bring the exponent down in front as a multiplier, and then decrease the exponent by exactly one.

  • Applies to any real number n (positive, negative, or fractional).
  • The variable x must be the base.

Core Formula

If f(x) = x^n

f'(x) = n · xn-1

edit_square Worked Example

Problem:

Find the derivative of f(x) = 3x4

  1. 1

    Identify components

    Constant multiple = 3. Base = x. Exponent (n) = 4.

  2. 2

    Apply Power Rule

    Bring down the 4 and multiply it by the constant 3.

    f'(x) = 3 · (4x4-1)

  3. 3

    Simplify

    f'(x) = 12x3
